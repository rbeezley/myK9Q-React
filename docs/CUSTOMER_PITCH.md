# myK9Q: The Complete Dog Sport Trial Management Platform

## Transform Your Trial Day Experience

myK9Q is a modern, offline-first platform designed specifically for AKC, UKC, and ASCA dog sport trials. Whether you're a trial secretary, judge, steward, or exhibitor, myK9Q streamlines every aspect of trial management with intelligent automation and real-time updates.

## Why myK9Q?

### **Built for the Real World**
- **Works Offline**: No WiFi at the trial site? No problem. myK9Q works completely offline and syncs when you reconnect.
- **Progressive Web App**: Install on any device - phone, tablet, or computer. No app store required.
- **Real-time Updates**: Changes sync instantly across all devices when online. Everyone stays on the same page.

### **Smart Trial Management**

**For Trial Secretaries & Stewards:**
- **Multi-Organization Support**: Manage AKC, UKC, and ASCA trials in one platform
- **Intelligent Check-In System**:
  - Track entry status (Not Checked In → Checked In → At Gate → In Ring)
  - Optional self-check-in for exhibitors
  - Visual status indicators and filtering
- **Automated Placement Calculation**: Scoring results automatically calculate placements with proper tie-handling
- **Class Requirements Engine**: Configurable rules for time warnings, minimum times, and scoring logic
- **Conflict Detection**: Automatically identifies scheduling conflicts for dogs entered in multiple classes

**For Judges:**
- **Organization-Specific Scoresheets**:
  - AKC Scent Work (multi-area timing)
  - AKC FastCat (speed scoring with MPH calculation)
  - UKC Obedience & Rally
  - ASCA Scent Detection
- **Multi-Timer Support**: Run multiple timers simultaneously for back-to-back runs
- **Auto-Save Protection**: Never lose scoring data - drafts save automatically every few seconds
- **Nationals Support**: Special scoring logic for national-level competitions

**For Exhibitors:**
- **Self-Service Check-In**: Check in your dogs when you arrive (if enabled by trial secretary)
- **Real-Time Class Status**: See live updates on class progress and your position in the running order
- **Entry History**: Track all your entries and results across multiple trials

### **Designed for Speed & Reliability**

**Performance:**
- Lightning-fast load times (< 2 seconds on most devices)
- Optimized database views for instant class summaries
- Responsive design works perfectly on phones, tablets, and desktops

**Security & Multi-Tenancy:**
- **License Key System**: Each trial gets a unique license key for complete data isolation
- **Role-Based Access**: Four permission levels (Admin, Judge, Steward, Exhibitor) with passcode authentication
- **No Passwords**: Simple passcode format (e.g., `aa260` for admin) derived from license key

**Data Management:**
- **Automatic Cleanup**: Trial data older than 30 days auto-archives to keep the system fast
- **Offline Queue**: All changes tracked and synced when connection restored
- **Draft Recovery**: Multiple auto-saved drafts per entry prevent data loss

### **Professional Features**

**Analytics & Reporting:**
- Class summary statistics (total entries, completed runs, qualified runs)
- Trial-level dashboards with real-time progress
- Performance views for instant data aggregation

**Smart Notifications:**
- Push notifications for class status changes
- Connection warnings when using cellular data
- Visual indicators for sync status

**Flexible Configuration:**
- Configurable class requirements (time warnings, minimum times)
- Custom competition types (Regular, National, Invitational)
- Organization-specific scoring rules
