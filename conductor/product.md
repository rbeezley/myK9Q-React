# Product Guide - myK9Q

## 1.0 Product Vision
myK9Q is a professional, production-ready Progressive Web App (PWA) designed to modernize dog show ring scoring and management. It provides a seamless, real-time experience for judges, stewards, and exhibitors, even in remote venues with limited connectivity.

## 2.0 Target Audience
- **Judges:** Perform real-time scoring with specialized digital scoresheets.
- **Stewards:** Manage entries, check-ins, and ring flow.
- **Trial Secretaries/Admins:** Oversee show setup, license management, and overall trial coordination.
- **Exhibitors:** View live scores and entry status for their dogs.

## 3.0 Core Values & Goals
- **Reliability:** Offline-first architecture ensures scoring never stops due to network issues.
- **Accuracy:** Automated calculations and validation logic for complex organization rules (AKC, UKC, ASCA).
- **Efficiency:** Real-time synchronization reduces manual data entry and speeds up trial results.
- **Sophistication:** A modern, touch-optimized, Apple-inspired UI providing a professional experience.

## 4.0 Key Features
- **Multi-Organization Support:** Specialized scoring logic for AKC (Scent Work, FastCat), UKC (Obedience, Rally, Agility), and ASCA (Scent Detection).
- **Offline-First Scoring:** Robust local storage and background synchronization.
- **Role-Based Access:** Secure passcode-driven authentication tailored to user roles.
- **Real-Time Dashboard:** Instant updates across all connected devices for check-ins and scores.
- **Automated Passcode System:** Intelligent credential generation based on show license keys.

## 5.0 Success Metrics
- **Zero Data Loss:** Successful synchronization of all offline scores once connectivity is restored.
- **User Efficiency:** Reduction in time required to finalize class results compared to paper scoring.
- **System Stability:** High performance and reliability across various mobile devices and network conditions.
